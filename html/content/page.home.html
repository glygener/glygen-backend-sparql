Overview Diagram
Data model represents the logical structure of data integration in GlyGen. It provides the definition, structure and format of the data stored and used in GlyGen database. The data is divided into three main categories: Protein, Proteoform and Glycan. These categories are represented as the main cores in the data model and their integration is described through a collection of predicates over a finite set of predicate variables. This network can be viewed using WebVOWL which is web application for the interactive visualization of ontologies.
